PDS Final Project
=================

Welcome to the documentation of the PDS Final Project.

1. Introduction

The PDS Final Project is focused on question classification and answer processing using Machine Learning models. The project includes:

Processing JSON-based questionnaires

Classifying questions and answers using a fine-tuned transformer model

Leveraging LLMs (Large Language Models) for improved categorization

This documentation provides an overview of the project, including installation, data processing, training, and inference.
